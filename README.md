## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

### Functional

Virtual kanban.
Authorized user can create Boards, tasks, related to 
certain board. And assign other users(or assign to himself) for task.
All task have 4 statuses, named by progress of the task.

soon different roles & rights will be added.
